# Changelog

## 2026-06-23

- 修复部分电脑翻译失败：`无法连接到远程服务器`。
- 翻译新增备用接口，主接口不可达时自动切换。
- 翻译请求新增 8 秒超时，避免长时间卡住。

## 2026-06-22

- OCR 结果页新增 `译英` 和 `译中`。
- 翻译从逐行请求优化为整段一次请求，响应更快。
- 新增翻译缓存，同一段文字重复翻译会直接显示缓存结果。
- 程序优先使用安装包内自带的 `Tesseract-OCR\tesseract.exe`，便携包无需额外安装 Tesseract。
- 新增傻瓜式便携安装包：解压后运行 `Install.bat` 即可安装并启动。
